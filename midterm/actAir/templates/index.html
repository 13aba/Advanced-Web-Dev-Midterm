Air quality in ACT community

API links

  • POST 127.0.0.1:8000/air-quality/ - show every measurement will also accept data from user at the bottom

    127.0.0.1:8000/air-quality/

  • GET 127.0.0.1:8000/location/{id} - show location informations like id, name, and gps

    127.0.0.1:8000/location/

  • GET 127.0.0.1:8000/air-quality/{location} - Show only given locations latest air measurements

    127.0.0.1:8000/air-quality/1

  • GET 127.0.0.1:8000/air-quality/history/{location}?date={date} - show locations air quality history depending on given date. Limited to only last 5 year . Date is written in YYYYMMDD format

    127.0.0.1:8000/air-quality/history/2?date=20220526

  • GET 127.0.0.1:8000/air-quality/average/average?start_date={start}&end_date={end} - show average air quality across all of the location. Date is written in YYYYMMDD format

    127.0.0.1:8000/air-quality/average/average?start_date=20230501&end_date=20230402

  • GET 127.0.0.1:8000/air-quality/element/{element}?start_date={start}&end_date={end} - show speficic elements history. Date is written in YYYYMMDD format. Possible elements are CO2 O3 NO2 PM10 PM2_5

    127.0.0.1:8000/air-quality/element/CO2?start_date=20230501&end_date=20230402
